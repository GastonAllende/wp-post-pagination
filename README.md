# Wp-Post-Pagination

Get data with WP rest api and present posts

## Built With

- Vanilla JS
- SASS

## Getting Started

Please follow the instructions below.

### Installing

Clone the repo

```
git clone https://github.com/GastonSaavedra/YourTalent.git
```

Install Dependencies

```
npm install
```

Run the page

```
npm run dev
```
